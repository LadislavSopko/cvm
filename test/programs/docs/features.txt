CVM Feature List

Language Features:
- Variables and basic types (string, number, boolean, null, undefined)
- Arrays with push, length, and element access
- Objects with property access and modification
- Control flow: if/else, while, for-of with break/continue
- String methods: substring, indexOf, split, slice, charAt, toUpperCase, toLowerCase
- JSON.parse and JSON.stringify
- File system operations via fs.listFiles()

Special Features:
- CC (Cognitive Call) for AI task creation
- Sandboxed file system access
- State persistence across CC calls
- MCP (Model Context Protocol) server integration
